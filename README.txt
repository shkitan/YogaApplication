** Yoga Pose Classification and Correction Android Application **
This repository contains the files for an Android application designed to classify and correct yoga poses. The application uses machine learning models to analyze user poses and provide real-time feedback for improvement.
You can see the Classification and Correction models file in repo named yoga-pose-classification-and-correction.

Project Overview
This project includes:

An Android application built in Android Studio.
A Flask server (flask_server.py) that handles the backend processing for pose classification and correction.

Features
Real-time Yoga Pose Classification: Uses the device's camera to capture the user's pose and classify it into predefined yoga poses.
Pose Correction: Provides feedback to users to help them correct their poses for better alignment and effectiveness.
User-Friendly Interface: Designed to be intuitive and easy to use for users of all skill levels.
